
## Installation

Install my-project with npm

```bash
  npm i
  cd mylargeReactapp
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/pritmodi98/mylargeReactapp.git
```

Go to the project directory

```bash
  cd mylargeReactapp
```

Install dependencies

```bash
  npm install
```

Start the frontEnd server (http://localhost:3000)


```bash
  npm start
```
